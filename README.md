# Amazon Review Helpfulness Analysis

This project analyzes what makes an Amazon product review genuinely helpful to other shoppers. I designed and built this project to explore how writing style, emotional tone, authenticity, and behavioral signals influence review helpfulness using real-world e-commerce data.

The goal was to create something practical and realistic that shows how data analytics, NLP, and psycholinguistics can support better decision-making on e-commerce platforms.

The project analyzes Amazon consumer electronics reviews (2019‚Äì2025) to identify which linguistic and behavioral factors drive helpfulness votes. It examines review length, sentiment, sarcasm, humor, emotional tone, verification status, and rating score.
